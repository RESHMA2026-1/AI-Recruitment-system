# 🤖 AI Recruitment System

## 📌 Overview
The AI Recruitment System is designed to automate resume analysis and candidate evaluation using intelligent parsing modules.

It extracts structured information such as education, certifications, skills, and experience from resumes and helps in candidate screening.

---

## 🚀 Features
- 📄 Resume text extraction
- 🎓 Education & certification detection
- 🧠 Data normalization
- 📊 Structured candidate profile generation
- 🔍 Job description parsing
- ⚖️ Candidate-job relevance analysis

---

## 🧩 Modules

### 1. Resume Parser
- Extracts raw text from resumes
- Handles formatting issues and noise

### 2. Education Parser
- Detects:
  - Degree
  - Field of study
  - Institution
  - Graduation year
- Extracts certifications
- Normalizes academic data
- Tags certification categories

### 3. JD Parser
- Extracts job requirements
- Identifies required skills and qualifications
