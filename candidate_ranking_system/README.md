# 🤖 AI Recruitment System

## 📌 Overview
The AI Recruitment System is designed to automate and enhance the hiring process by analyzing resumes, extracting relevant information, and ranking candidates based on job requirements. The system ensures fairness, reduces bias, and improves decision-making.

---

## 🎯 Objectives
- Automate resume screening
- Extract structured information from resumes
- Match candidates with job descriptions
- Rank candidates based on relevance
- Ensure fairness and reduce bias in evaluation

---

## 🧩 Modules

### 1. 📄 Resume Parsing (Education & Certificate Extraction)
- Extracts education details from resumes
- Identifies certifications and qualifications
- Converts unstructured data into structured format

---

### 2. 🧠 Semantic Matching Engine
- Matches candidate profiles with job descriptions
- Uses similarity techniques instead of keyword matching
- Improves accuracy in candidate selection

---

### 3. 💼 Experience Parsing & Relevance Engine
- Extracts work experience from resumes
- Evaluates relevance to job roles
- Calculates experience-based scores

---

### 4. 🧮 Candidate Ranking System
- Assigns scores to candidates
- Ranks candidates based on performance
- Applies shortlisting rules:
  - Shortlisted
  - Review
  - Rejected

---

### 5. ⚖️ Fairness, Normalization & Bias Reduction
- Normalizes resume data into standard format
- Masks sensitive attributes (name, gender)
- Reduces bias in evaluation
- Normalizes scores (0–1 range)
- Ensures fair comparison between candidates

---

## ⚙️ Technologies Used
- Python
- Basic NLP techniques
- Data processing using dictionaries & lists
